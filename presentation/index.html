User Stories & Features

A walkthrough of every major feature with real screenshots

1. Authentication

Secure login with email/password, registration with email verification, password reset, and two-factor authentication (TOTP).

2. Organization Setup Wizard

After registration, a 4-step guided wizard configures the business: company info, primary location, currency & tax settings, and contact details.

2. Setup Wizard (continued)

Multi-Country India GST UAE VAT US Sales Tax EU VAT

3. Dashboard

Business overview with total invoiced, collected, outstanding amounts, invoice status breakdown, 6-month trend chart, top customers, overdue invoices, and recent activity.

4. Organization Management

Manage business details across multiple tabs: basics, location, bank details, and contacts. Support for multiple locations per organization.

5. Customer Management

Full CRUD for customers with multiple contacts, multi-currency support, location management, and search/filter. Customers with existing invoices are protected from deletion.

6. Invoice Creation

Create invoices with line items, automatic tax calculations, customer selection with auto-populated locations, and real-time total preview.

6. Invoice Editing & Actions

Edit existing invoices, send via email with PDF attachment, download PDF, view public link, duplicate, and mark as paid.

7. Estimates

Create estimates with the same powerful form as invoices. One-click conversion from estimate to invoice preserves all data.

8. Public Invoice & Estimate Views

Professional, shareable document views accessible via unique ULID links without authentication. Clients can view and print directly from the browser.

8. Invoice Status Indicators

Public views show clear status badges: draft, sent, paid, overdue. Overdue invoices display prominently with the amount due.

9. Invoice Numbering Series

Flexible numbering system with format tokens ({PREFIX}, {YEAR}, {SEQUENCE:4}, {FY}), reset frequencies (never, monthly, yearly, financial year), location-specific series, and automatic fallback to defaults.

10. PDF Generation

Professional A4 PDFs generated via Gotenberg with full Chromium rendering. Supports organization logos, tax breakdowns, and multi-currency formatting.

Architecture & Infrastructure

Production-grade deployment with Docker

Application Architecture

Domain-Driven Design

  • Value Objects (ContactCollection, InvoiceTotals)
  • Service Layer (InvoiceCalculator, PdfService)
  • Rich domain models with business methods
  • Custom casts for JSON columns

Multi-Tenant Data Model

  • Organization-centric (teams renamed to orgs)
  • Polymorphic Location model
  • ULID identifiers for public sharing
  • OrganizationScope for data isolation

Frontend (SPA)

  • Inertia.js v3 (no API layer needed)
  • Vue 3 Composition API + TypeScript
  • Wayfinder for type-safe route references
  • Real-time calculations in forms

Docker Infrastructure

  • Dev: Laravel Sail (7 services)
  • Prod: ServerSideUp FrankenPHP + Octane
  • Gotenberg for PDF generation
  • Production image testing in CI
